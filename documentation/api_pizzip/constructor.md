layout: default
section: api
---

__Description__ : Create a new PizZip instance.

__Arguments__ : None

__Returns__ : A new PizZip.

__Throws__ : Nothing.

<!-- __Complexity__ : Object creation in **O(1)**. -->

__Example__

```js
var zip = new PizZip();
// same as
var zip = PizZip();
```
